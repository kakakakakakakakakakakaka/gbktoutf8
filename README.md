>项目依赖[cpdetector](http://cpdetector.sourceforge.net/index.shtml)
>>如需自行编译源码请导入 **cpdetector**,**antlr**,**chardet**,**jargs**

### 如何使用

点击`浏览`选择需要转码的路径，单选，可选择`文件`或`文件夹`，确认路径后点击`转码`即可
![](https://i.loli.net/2019/04/12/5cafd878b4480.png)
